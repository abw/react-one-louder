## Themed Component

Use the `Themed` function to wrap your component.  Give it a unique name so
you can reference it later.

In this example, the default `volume` for the `Amplifier` component is 10.

## Theme Provider

You can then use the `Theme.Provider` as a wrapper around your application
and provide it with new default properties for your components.

Here we change the default volume for the `Amplifier` component to 11.
It's one louder.  Note that this only defines the default properties.  If
you explicitly pass the `volume` property to a component then it'll use
that instead.
